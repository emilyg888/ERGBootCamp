# ERGBootCamp — Weekly Review Prompt

Review the past 7 days of training data and provide:

1. **Volume summary** — total meters, total minutes, sessions completed
2. **Performance trend** — split direction, consistency (std dev), best/worst session
3. **Recovery alignment** — did Garmin recovery signals correlate with performance?
4. **Load assessment** — was weekly load appropriate for current training phase?
5. **Technique observation** — any patterns in stroke rate or split variance?
6. **Next week prescription** — focus, volume, key session
7. **Competition readiness** — % progress toward race-day targets

Reference the coaching memory for context on intentional recovery rows.
